 # <center>学习逻辑</center>

 ## 优秀，也要努力

 优秀的人如果不够努力，必然竞争不过天赋一般却十分勤奋的人。

 ## 不要表演“努力”

 真正的努力不是比谁把自己虐的更惨，也不是比谁花的时间更多，而是做对事，做好事，用有效的行动让每一天看到效果。

 ## 机会需要争取，也需要储备

 - 我们一生中遇到的大部分机会并不能立马变现，但却是一种极为重要的储备。
 - 人必须逼自己一把，给自己一个于强者竞争、向更优胜者挑战的机会。

## 学历代表过去，只有学习力才能代表将来

学习力，是对未知的知识主动汲取和积极上进的能力。

- 学习的动力。
- 学习的毅力。
- 学习的能力。
